Etiquetas basicas de HTLM atributos y valores

El proposito de esta primera pagina web es aque el cual permite conocer los propositos de los atributos de html dando una introduccion a este mundo junto con las definiciones de cada una de las mencionadas anteriormente

Etiquetas

Segun solaya al definir el codigo html como un lenguaje de etiquetas o marcas los elementos que pasan a conformar una pagina web se describen mediante palabras especiales que marcan el inicio y el final de los mismos, genertalmente estas siempre vienen en pares aunque existen excepciones a las reglas. Fuente
Acontinuacion daremos ejemplos de etiquetas usadas para crear esta pagina.

La etiqueta "!DOCTYPE html" Es aquella que define el archivo como un documento html

La etiqueta "html" Es aquel que encierra o abarca todo el contenido del documento.

La etiqueta "head" Esta etiqueta es un contenedor de todo aquello que se quiere incluir en la pagina HTML y no es visible para los visitantes

La etiqueta "title" Esta estable el titulo dde la pagina que se desarrollo, en este caso "Etiquetas basicas web"

La etiqueta "body" Es la etiqueta que encierra todo el contenido de esta pagina con la intencion de mostrar a los usuarios dicha informacion

Atributos

Segun los documentos de desarrollo de mozilla los atributos son aquellos valores que configuran elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios.
